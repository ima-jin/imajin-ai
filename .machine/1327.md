# Epic: Autonomous sprint loop (karaoke first tenant)

Loop: PO agent drafts 5 features/sprint → ranks 3 → human approves picks →
coder agents (gh-issues --cron) build → CI (tests/build/Sonar) → human gates
per-PR [Merge to dev]/[Iterate]/[Kill]. cron fires next tick.

Invariants:
- PO proposes, never merges.
- Human gate on merge-to-dev. Merge = to main → auto-deploy dev only.
- Never straight to prod.

Milestones:
- M1: cron→PO subagent→draft 5/rank 3→Telegram approve→gh-issues --cron→CI→digest gate.
- M2: observe one full cycle; measure Sonar delta, review burden, correctness.
- M3: wire each loop action as signed event (#970/#633).
- M4 (gated on M2): selectively auto-merge CI+rubric-pass non-behavioral PRs.

Reuses: gh-issues skill (swarm/--cron/--fork), CI, sessions_spawn, cron.
